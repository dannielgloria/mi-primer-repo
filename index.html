Fundamentos de HTML

HTML es el lenguaje de marcado utilizado para crear páginas web. Es la base de cualquier sitio web y se utiliza para estructurar el contenido, como texto, imágenes, enlaces y otros elementos multimedia.

Anatomía de un elemento HTML

Un elemento HTML consta de una etiqueta de apertura, contenido y una etiqueta de cierre. Por ejemplo:

<h1>Título principal</h1>

En este caso, <h1> es la etiqueta de apertura, "Título principal" es el contenido y </h1> es la etiqueta de cierre.

Los atributos proporcionan información extra. Ejemplo: class="nota"

Contenido en HTML

El contenido en HTML puede ser texto, imágenes, enlaces, listas, tablas y otros elementos multimedia. Es importante estructurar el contenido de manera clara y organizada para mejorar la legibilidad y la experiencia del usuario.
